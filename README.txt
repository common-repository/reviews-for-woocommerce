This plugin provides different template to show WooCommerce reviews of any product.

== Description ==

**Reviews for WooCommerce** plugin displays WooCommerce product reviews in slider. You just have to paste the shortcode in the page with some attributes.
 
Shortcode for displaying review slider for product id=12345 is [revfwoo_review id="12345" style="slider"].

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Go to Add New section of Plugins Menu and search for 'Reviews for WooCommerce'.
2. Now you can install the Reviews for WooCommerce Plugin.
3. Activate the plugin for displaying WooCommerce product review.

== Frequently Asked Questions ==

= Can we use this shortcode in any page =

Yes you can simply paste the shortcode with given attributes in any page to display slider reviews of any product.

= How many reviews will be displayed on the slider? =

Currently 5 reviews can be displayed in slider.

= Is there any prerequisite to use that plugins? =

Yes to use this plugin we need to WooCommerce plugin active in wordpress.

= How to use this plugin? =

Check the plugin information tab from the "Reviews for Woo" menu where you can see the How to use section.
